# (WIP) LELA
## An Erlang REST Server, powered by MongoDB

The server is runnin on port 4500, and the client, a VueJS client, is running on a default port of 8080.
Change them as you will.

# Server
```
make
sh run.sh
```

# Client
```
cd client
npm run dev
```
